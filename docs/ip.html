Comment ça marche ?

  1. Dans l'application, appuyez sur 📤 Partager le rapport
  2. L'application génère un rapport complet : IPs, speedtest, DNS leak, test de blocage
  3. Le rapport est envoyé sur tmpfiles.org et un numéro est copié dans le presse-papier
  4. Envoyez ce numéro à la personne qui en a besoin
  5. Elle vient sur cette page, colle le numéro, et consulte le rapport

⏰ Les rapports expirent automatiquement après 60 minutes.
