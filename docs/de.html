Funktionen

🌐

DNS DoH / DoQ / DoT / Standard

Unterstützt DNS over HTTPS, DNS over QUIC (DoQ), DNS over TLS (ADB/Shizuku) und klassisches UDP-DNS.

⚡

Aktivierung mit einem Klick

Wählen Sie einen DNS-Anbieter und aktivieren Sie ihn sofort über lokales VPN oder Private DNS.

📊

DNS-Geschwindigkeitstest

Testen Sie die Latenz aller Ihrer DNS-Anbieter und finden Sie den schnellsten für Sie.

🔀

URL-Umleitung

Leiten Sie Domains auf andere um, um DNS-Sperren Ihres Anbieters zu umgehen.

🔄

Automatische Wiederverbindung

Automatische DNS-Wiederverbindung beim Gerätestart. Unterstützt auch IPv6-Umschaltung.

💾

Import / Export

Sichern und Wiederherstellen Ihrer Konfiguration. Ende-zu-Ende-verschlüsselter Export und Import (AES-256-GCM).

📡

Shizuku / ADB

Shizuku-Unterstützung zum Ändern von Private DNS ohne PC. Kompatibel mit Android TV & Firestick.

🌍

Mehrsprachig

Oberfläche verfügbar in 12 Sprachen: Französisch, Englisch, Spanisch, Italienisch, Portugiesisch, Russisch, Chinesisch, Arabisch, Hindi, Bengalisch, Japanisch und Deutsch.

🔄

Automatisches Update

Automatische Update-Prüfung beim Start. Download und Installation direkt in der App.

🛡

Split Tunneling

Bestimmte Apps vom DNS-VPN ausschließen. Nützlich für inkompatible Apps.

🔍

Schneller DNS-Test

DNS-Leak-Test und Domain-Name-Test mit einem Klick. Überprüfen Sie Ihren Schutz in Echtzeit.
