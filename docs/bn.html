🌐

DNS DoH / DoT / Standard

DNS over HTTPS (লোকাল VPN), DNS over TLS (ADB/Shizuku), এবং ক্লাসিক UDP DNS সমর্থন করে।

⚡

এক-ক্লিকে সক্রিয়করণ

একটি DNS প্রদানকারী নির্বাচন করুন এবং লোকাল VPN বা Private DNS এর মাধ্যমে তাৎক্ষণিকভাবে সক্রিয় করুন।

📊

DNS স্পিডটেস্ট

আপনার সমস্ত DNS প্রদানকারীর লেটেন্সি পরীক্ষা করুন এবং আপনার জন্য সবচেয়ে দ্রুততম খুঁজে বের করুন।

⭐

পছন্দসই এবং ডিফল্ট DNS

পছন্দসই দিয়ে আপনার পছন্দের DNS সার্ভারগুলি সংগঠিত করুন এবং একটি ডিফল্ট DNS সেট করুন।

🔀

URL পুনর্লিখন

আপনার ISP-এর DNS ব্লক এড়াতে ডোমেইনগুলি অন্যদের দিকে রিডাইরেক্ট করুন।

🔄

স্বয়ংক্রিয়-পুনঃসংযোগ

ডিভাইস বুটে স্বয়ংক্রিয় DNS পুনঃসংযোগ। IPv6 টগলও সমর্থন করে।

💾

আমদানি / রপ্তানি

আপনার সম্পূর্ণ কনফিগারেশন ব্যাকআপ এবং পুনরুদ্ধার করুন। উপাদান নির্বাচনসহ বিস্তারিত রপ্তানি।

📡

Shizuku / ADB

PC ছাড়াই Private DNS পরিবর্তন করতে Shizuku সমর্থন। Android TV এবং Firestick সামঞ্জস্যপূর্ণ।

🌍

বহুভাষিক

11টি ভাষায় উপলব্ধ: ফরাসি, ইংরেজি, স্প্যানিশ, ইতালীয়, পর্তুগিজ, রাশিয়ান, চীনা, আরবি, হিন্দি, বাংলা এবং জাপানি।

🔄

স্বয়ংক্রিয়-আপডেট

লঞ্চে স্বয়ংক্রিয় আপডেট চেক। অ্যাপের মধ্যেই ডাউনলোড এবং ইনস্টলেশন।
