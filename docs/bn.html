বৈশিষ্ট্যসমূহ

🌐

DNS DoH / DoT / DoQ / Standard

DNS over HTTPS (লোকাল VPN), DNS over TLS (ADB/Shizuku), DNS over QUIC (DoQ), এবং ক্লাসিক UDP DNS সমর্থন করে।

⚡

এক-ক্লিকে সক্রিয়করণ

একটি DNS প্রদানকারী নির্বাচন করুন এবং লোকাল VPN বা Private DNS এর মাধ্যমে তাৎক্ষণিকভাবে সক্রিয় করুন।

📊

DNS স্পিডটেস্ট

আপনার সমস্ত DNS প্রদানকারীর লেটেন্সি পরীক্ষা করুন এবং আপনার জন্য সবচেয়ে দ্রুততম খুঁজে বের করুন।

🔀

URL পুনর্লিখন

আপনার ISP-এর DNS ব্লক এড়াতে ডোমেইনগুলি অন্যদের দিকে রিডাইরেক্ট করুন।

🔄

স্বয়ংক্রিয়-পুনঃসংযোগ

ডিভাইস বুটে স্বয়ংক্রিয় DNS পুনঃসংযোগ। IPv6 টগলও সমর্থন করে।

💾

আমদানি / রপ্তানি

আপনার সম্পূর্ণ কনফিগারেশন ব্যাকআপ এবং পুনরুদ্ধার করুন। উপাদান নির্বাচনসহ বিস্তারিত রপ্তানি। AES-256-GCM এনক্রিপশন।

📡

Shizuku / ADB

PC ছাড়াই Private DNS পরিবর্তন করতে Shizuku সমর্থন। Android TV এবং Firestick সামঞ্জস্যপূর্ণ।

🌍

বহুভাষিক

12টি ভাষায় উপলব্ধ: ফরাসি, ইংরেজি, স্প্যানিশ, ইতালীয়, পর্তুগিজ, রাশিয়ান, চীনা, আরবি, হিন্দি, বাংলা, জাপানি এবং জার্মান।

🔄

স্বয়ংক্রিয়-আপডেট

লঞ্চে স্বয়ংক্রিয় আপডেট চেক। অ্যাপের মধ্যেই ডাউনলোড এবং ইনস্টলেশন।

🛡

Split Tunneling

নির্দিষ্ট অ্যাপগুলি DNS VPN থেকে বাদ দিন। অসঙ্গত অ্যাপের জন্য উপযোগী।

🔍

দ্রুত DNS পরীক্ষা

এক ক্লিকে DNS লিক পরীক্ষা এবং ডোমেন নাম পরীক্ষা। রিয়েল টাইমে আপনার সুরক্ষা যাচাই করুন।
