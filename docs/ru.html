Функции

🌐

DNS DoH / DoQ / DoT / Standard

Поддержка DNS over HTTPS, DNS over QUIC (DoQ), DNS over TLS (ADB/Shizuku) и классического UDP DNS.

⚡

Активация в один клик

Выберите DNS-провайдера и мгновенно активируйте его через локальный VPN или Private DNS.

📊

DNS Speedtest

Проверьте задержку всех ваших DNS-провайдеров и найдите самый быстрый для вас.

🔀

URL Rewrite

Перенаправляйте домены на другие, чтобы обойти DNS-блокировки вашего провайдера.

🔄

Автоподключение

Автоматическое подключение к DNS при загрузке устройства. Также поддерживает переключение IPv6.

💾

Импорт / Экспорт

Сохраняйте и восстанавливайте конфигурацию. Шифрованный экспорт и импорт (AES-256-GCM).

📡

Shizuku / ADB

Поддержка Shizuku для изменения Private DNS без ПК. Совместимость с Android TV и Firestick.

🌍

Мультиязычность

Интерфейс доступен на 12 языках: французский, английский, испанский, итальянский, португальский, русский, китайский, арабский, хинди, бенгальский, японский и немецкий.

🔄

Автообновление

Автоматическая проверка обновлений при запуске. Загрузка и установка прямо из приложения.

🛡

Split Tunneling

Исключите определённые приложения из DNS VPN. Полезно для несовместимых приложений.

🔍

Быстрый тест DNS

Тест утечки DNS и тест доменных имён в один клик. Проверьте свою защиту в реальном времени.
