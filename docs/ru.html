Функции

🌐

DNS DoH / DoT / Стандартный

Поддержка DNS over HTTPS (локальный VPN), DNS over TLS (ADB/Shizuku) и классического DNS через UDP.

⚡

Активация в один клик

Выберите DNS-провайдера и мгновенно активируйте его через локальный VPN или Private DNS.

📊

DNS Speedtest

Проверьте задержку всех ваших DNS-провайдеров и найдите самый быстрый для вас.

⭐

Избранное и DNS по умолчанию

Организуйте любимые DNS-серверы с помощью избранного и задайте DNS по умолчанию.

🔀

URL Rewrite

Перенаправляйте домены на другие, чтобы обойти DNS-блокировки вашего провайдера.

🔄

Автоподключение

Автоматическое подключение к DNS при загрузке устройства. Также поддерживает переключение IPv6.

💾

Импорт / Экспорт

Сохраняйте и восстанавливайте полную конфигурацию. Гранулярный экспорт с выбором элементов.

📡

Shizuku / ADB

Поддержка Shizuku для изменения Private DNS без ПК. Совместимость с Android TV и Firestick.

🌍

Мультиязычность

Интерфейс доступен на 11 языках: французский, английский, испанский, итальянский, португальский, русский, китайский, арабский, хинди, бенгальский и японский.

🔄

Автообновление

Автоматическая проверка обновлений при запуске. Загрузка и установка прямо из приложения.

Встроенные DNS-провайдеры

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark Yandex + пользовательские профили
