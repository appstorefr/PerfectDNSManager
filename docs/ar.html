الميزات

🌐

DNS DoH / DoT / قياسي

يدعم DNS over HTTPS (VPN محلي)، DNS over TLS (ADB/Shizuku)، و DNS الكلاسيكي عبر UDP.

⚡

تفعيل بنقرة واحدة

اختر مزود DNS وقم بتفعيله فورياً عبر VPN محلي أو Private DNS.

📊

اختبار سرعة DNS

اختبر زمن الاستجابة لجميع مزودي DNS لديك وابحث عن الأسرع بالنسبة لك.

⭐

المفضلة و DNS الافتراضي

نظّم خوادم DNS المفضلة لديك باستخدام المفضلة وحدد DNS افتراضي.

🔀

إعادة كتابة URL

أعد توجيه النطاقات إلى أخرى لتجاوز حظر DNS من مزود الخدمة.

🔄

إعادة الاتصال التلقائي

إعادة اتصال تلقائية بـ DNS عند تشغيل الجهاز. يدعم أيضاً تبديل IPv6.

💾

استيراد / تصدير

احفظ واستعد الإعدادات الكاملة. تصدير دقيق مع اختيار العناصر.

📡

Shizuku / ADB

دعم Shizuku لتعديل Private DNS بدون حاسوب. متوافق مع Android TV و Firestick.

🌍

متعدد اللغات

الواجهة متوفرة بـ 8 لغات: الفرنسية والإنجليزية والإسبانية والإيطالية والبرتغالية والروسية والصينية والعربية.

🔄

تحديث تلقائي

فحص تلقائي للتحديثات عند التشغيل. تنزيل وتثبيت من داخل التطبيق.

مزودو DNS المدمجون

أكثر من 15 مزود DNS مُعد مسبقاً، مع تصنيف السرعة والخصوصية :

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark Yandex + ملفات تعريف مخصصة
