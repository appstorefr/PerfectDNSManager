الميزات

🌐

DNS DoH / DoT / DoQ / قياسي

يدعم DNS over HTTPS، DNS over TLS، DNS over QUIC (DoQ) و DNS الكلاسيكي عبر UDP.

⚡

تفعيل بنقرة واحدة

اختر مزود DNS وقم بتفعيله فورياً عبر VPN محلي أو Private DNS.

📊

اختبار سرعة DNS

اختبر زمن الاستجابة لجميع مزودي DNS لديك وابحث عن الأسرع بالنسبة لك.

🔀

إعادة كتابة URL

أعد توجيه النطاقات إلى أخرى لتجاوز حظر DNS من مزود الخدمة.

🔄

إعادة الاتصال التلقائي

إعادة اتصال تلقائية بـ DNS عند تشغيل الجهاز. يدعم أيضاً تبديل IPv6.

💾

استيراد / تصدير

احفظ واستعد الإعدادات الكاملة. تصدير دقيق مع اختيار العناصر. تشفير AES-256-GCM.

📡

Shizuku / ADB

دعم Shizuku لتعديل Private DNS بدون حاسوب. متوافق مع Android TV و Firestick.

🌍

متعدد اللغات

واجهة متوفرة بـ 12 لغة: الفرنسية والإنجليزية والإسبانية والإيطالية والبرتغالية والروسية والصينية والعربية والهندية والبنغالية واليابانية.

🔄

تحديث تلقائي

فحص تلقائي للتحديثات عند التشغيل. تنزيل وتثبيت من داخل التطبيق.

🛡

Split Tunneling

استبعد تطبيقات محددة من DNS VPN. مفيد للتطبيقات غير المتوافقة.

🔍

اختبار DNS سريع

اختبار تسرب DNS واختبار أسماء النطاقات بنقرة واحدة. تحقق من حمايتك في الوقت الفعلي.

مزودو DNS المدمجون →

أكثر من 15 مزود DNS مُعد مسبقاً، مع تصنيف السرعة والخصوصية :

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark FDN Yandex + ملفات تعريف مخصصة
