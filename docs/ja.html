機能

🌐

DNS DoH / DoT / Standard

DNS over HTTPS（ローカルVPN）、DNS over TLS（ADB/Shizuku）、従来のUDP DNSに対応。

⚡

ワンクリック有効化

DNSプロバイダーを選択し、ローカルVPNまたはPrivate DNSで即座に有効化。

📊

DNS速度テスト

すべてのDNSプロバイダーのレイテンシをテストし、最速のものを見つけます。

⭐

お気に入り＆デフォルトDNS

お気に入りでDNSサーバーを整理し、デフォルトDNSを設定できます。

🔀

URLリライト

ドメインを別のドメインにリダイレクトし、ISPのDNSブロックを回避します。

🔄

自動再接続

デバイス起動時にDNSを自動再接続。IPv6切り替えにも対応。

💾

インポート / エクスポート

設定全体のバックアップと復元。要素選択による詳細なエクスポート。

📡

Shizuku / ADB

PCなしでPrivate DNSを変更するShizukuサポート。Android TV＆Firestick対応。

🌍

多言語対応

11言語対応：フランス語、英語、スペイン語、イタリア語、ポルトガル語、ロシア語、中国語、アラビア語、ヒンディー語、ベンガル語、日本語

🔄

自動アップデート

起動時に自動でアップデートを確認。アプリ内ダウンロードとインストール。

内蔵DNSプロバイダー

15以上のDNSプロバイダーを事前設定済み、速度とプライバシーのランキング付き：

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark Yandex + カスタムプロファイル
