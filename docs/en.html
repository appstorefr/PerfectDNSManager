Features

🌐

DNS DoH / DoQ / DoT / Standard

Supports DNS over HTTPS, DNS over QUIC (DoQ), DNS over TLS (ADB/Shizuku), and classic UDP DNS.

⚡

One-click Activation

Select a DNS provider and activate it instantly via local VPN or Private DNS.

📊

DNS Speedtest

Test the latency of all your DNS providers and find the fastest one for you.

🔀

URL Rewrite

Redirect domains to others to bypass your ISP's DNS blocks.

🔄

Auto-reconnect

Automatic DNS reconnection on device boot. Also supports IPv6 toggle.

💾

Import / Export

Back up and restore your configuration. End-to-end encrypted export and import (AES-256-GCM).

📡

Shizuku / ADB

Shizuku support to change Private DNS without a PC. Android TV & Firestick compatible.

🌍

Multilingual

Interface available in 12 languages: French, English, Spanish, Italian, Portuguese, Russian, Chinese, Arabic, Hindi, Bengali, Japanese and German.

🔄

Auto-update

Automatic update check on launch. In-app download and installation.

🛡

Split Tunneling

Exclude specific apps from the DNS VPN. Useful for incompatible applications.

🔍

Quick DNS Test

DNS leak test and domain name test in one click. Verify your protection in real time.

Built-in DNS Providers →

More than 15 preconfigured DNS providers, with speed and privacy rankings:

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark FDN Yandex + custom profiles
