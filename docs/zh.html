功能特性

🌐

DNS DoH / DoQ / DoT / 标准

支持 DNS over HTTPS、DNS over QUIC (DoQ)（本地 VPN）、DNS over TLS（ADB/Shizuku）和经典 UDP DNS。

⚡

一键激活

选择 DNS 提供商，通过本地 VPN 或 Private DNS 即时激活。

📊

DNS 速度测试

测试所有 DNS 提供商的延迟，找到最适合您的最快服务器。

🔀

URL 重写

将域名重定向到其他域名，绕过运营商的 DNS 封锁。

🔄

自动重连

设备启动时自动连接 DNS。同时支持 IPv6 切换。

💾

导入 / 导出

保存和恢复配置。端到端加密导出和导入 (AES-256-GCM)。

📡

Shizuku / ADB

支持 Shizuku，无需电脑即可修改 Private DNS。兼容 Android TV 和 Firestick。

🌍

多语言

界面支持12种语言：法语、英语、西班牙语、意大利语、葡萄牙语、俄语、中文、阿拉伯语、印地语、孟加拉语、日语和德语。

🔄

自动更新

启动时自动检查更新。支持应用内下载和安装。

🛡

Split Tunneling

将特定应用排除在 DNS VPN 之外。适用于不兼容的应用。

🔍

快速 DNS 测试

一键 DNS 泄漏测试和域名测试。实时验证您的保护。
