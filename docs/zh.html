功能特性

🌐

DNS DoH / DoT / 标准

支持 DNS over HTTPS（本地 VPN）、DNS over TLS（ADB/Shizuku）和经典 UDP DNS。

⚡

一键激活

选择 DNS 提供商，通过本地 VPN 或 Private DNS 即时激活。

📊

DNS 速度测试

测试所有 DNS 提供商的延迟，找到最适合您的最快服务器。

⭐

收藏夹和默认 DNS

使用收藏夹整理您喜爱的 DNS 服务器，并设置默认 DNS。

🔀

URL 重写

将域名重定向到其他域名，绕过运营商的 DNS 封锁。

🔄

自动重连

设备启动时自动连接 DNS。同时支持 IPv6 切换。

💾

导入 / 导出

保存和恢复完整配置。支持精细导出，可选择导出内容。

📡

Shizuku / ADB

支持 Shizuku，无需电脑即可修改 Private DNS。兼容 Android TV 和 Firestick。

🌍

多语言

界面支持11种语言：法语、英语、西班牙语、意大利语、葡萄牙语、俄语、中文、阿拉伯语、印地语、孟加拉语和日语。

🔄

自动更新

启动时自动检查更新。支持应用内下载和安装。

内置 DNS 提供商

超过 15 个预配置的 DNS 提供商，含速度和隐私评级：

AdGuard Cloudflare ControlD dns.sb Google Mullvad NextDNS Quad9 Surfshark Yandex + 自定义配置
