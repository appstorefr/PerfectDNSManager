Funcionalidades

🌐

DNS DoH / DoT / Estándar

Compatible con DNS over HTTPS (VPN local), DNS over TLS (ADB/Shizuku) y DNS clásico UDP.

⚡

Activación en un clic

Selecciona un proveedor DNS y actívalo instantáneamente mediante VPN local o DNS Privado.

📊

DNS Speedtest

Prueba la latencia de todos tus proveedores DNS y encuentra el más rápido para ti.

⭐

Favoritos y DNS predeterminado

Organiza tus servidores DNS preferidos con favoritos y define un DNS predeterminado.

🔀

URL Rewrite

Redirige dominios a otros para eludir los bloqueos DNS de tu operador.

🔄

Auto-reconexión

Reconexión automática al DNS al iniciar el dispositivo. También soporta el toggle IPv6.

💾

Importar / Exportar

Guarda y restaura tu configuración completa. Exportación granular con selección de elementos.

📡

Shizuku / ADB

Soporte Shizuku para modificar DNS Privado sin PC. Compatible con Android TV & Firestick.

🌍

Multilingüe

Interfaz disponible en 11 idiomas: francés, inglés, español, italiano, portugués, ruso, chino, árabe, hindi, bengalí y japonés.

🔄

Actualización automática

Verificación automática de actualizaciones al iniciar. Descarga e instalación dentro de la app.
